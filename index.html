MATH PROBLEM

Fundamental Theorem of Calculus

If you integrate a function and then differentiate, you get the original function back: \[ \frac{d}{dx}\left( \int_a^x f(t)\, dt \right) = f(x). \]

To compute a definite integral, subtract the values of an antiderivative at the endpoints: \[ \int_a^b f(x)\, dx = F(b) - F(a), \] where \( F'(x) = f(x) \).
